1. 协程调度的时机

  • 使用关键字go

  • 内存同步访问

2. make和new的区别

  • make只适用于map/slice/channel, new用于内存分配, 并初始化零值
  • new不常用, make不可替代

3. 数组和切片的区别

  1. 数组是值类型, 切片是引用类型
  2. 数组初始化时需要指定长度, 切片不需要
  3. 数组是定长的, 切片是变长的
  4. 数组是切片存储数据的底层结构.
  5. 数组的长度是类型的一部分, 长度为20 的数组和我长度为10 的数组不是一个类型.

4. Golang中有哪些引用类型

  1. map
  2. slice
  3. channel
  4. interface

5. map对于key的类型有什么要求

不能是函数类型、字典类型、切片类型。
原因：go语言规范规定，键类型的值之间必须可以施加==操作符和！=操作符，即必须支持判等操作，由于函数类型，字典类型，切片类型不支持判等操作
